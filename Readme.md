# Тестовое задание БЮРО1440

## Описание ТЗ

Сделать простой конвертер данных.

- Две функции: одна для преобразования float в bytes, другая bytes в строку
- Должно быть логгирование операций и результата функций
- Результат второй функции от данных первой функции записать в файл

## Технические требования

Необходима установленная версия python 3.12 

## Запуск 

```bash
python converter.py # Windows
```

```bash
python3 converter.py # Linux/MacOS
```

## API документация

### Обзор

Модуль `converter` предоставляет три функции для преобразования чисел с плавающей точкой в байтовое представление и шестнадцатеричные строки, а также для сохранения результатов в файл.

### 1. float_to_bytes

Преобразует число с плавающей точкой в последовательность из 8 байт согласно стандарту IEEE 754 (double precision).

**Что делает:**
1. Проверяет тип входного значения
2. Упаковывает число в 8 байт
3. Логирует входной аргумент и результат
4. Возвращает байтовую последовательность

### 2. bytes_to_string

Преобразует байтовую последовательность в шестнадцатеричную строку.

**Что делает:**
1. Проверяет тип входного значения
2. Преобразует каждый байт в два шестнадцатеричных символа 
3. Логирует входной аргумент и результат
4. Возвращает hex-строку

### 3. save_float_conversion

Выполняет полный цикл конвертации и сохраняет результат в файл.

**Что делает:**

1. Проверяет тип входного значения
2. Вызывает float_to_bytes для получения байтов
3. Вызывает bytes_to_string для получения hex-строки
4. Создаёт папку logs/ (если не существует)
5. Генерирует уникальное имя файла на основе текущего времени
6. Записывает hex-строку в файл
7. Логирует результат
